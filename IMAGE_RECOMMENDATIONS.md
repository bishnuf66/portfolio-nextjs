## Best Practices

1. **Resolution**: Use high-resolution images (at least 2560x1440px) for the parallax effect
2. **File Format**: Use WebP for better performance, with JPG fallback
3. **File Size**: Optimize images to under 500KB using tools like:
   - TinyPNG: https://tinypng.com/
   - Squoosh: https://squoosh.app/

4. **Color Scheme**: Choose images that complement your brand colors (blue, purple, pink)
5. **Contrast**: Ensure text overlay is readable - darker images work better

## How to Replace

1. Download your chosen image
2. Optimize it using TinyPNG or Squoosh
3. Rename it (e.g., `coding-workspace.jpg`)
4. Place it in the `/public` folder
5. Update the image path in `src/components/Home.tsx`:
   ```tsx
   src="/your-image-name.jpg"
   ```
